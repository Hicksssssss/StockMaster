# StockMaster - Gestionnaire de stock

Gestionnaire de stock complet développé en **JavaFX + MySQL + JasperReports**.

## Fonctionnalités
- Authentification (login)
- Gestion des produits et du stock
- Gestion des clients
- Facturation en temps réel
- Enregistrement des achats et ventes
- Génération de rapports PDF (factures, listes clients, ventes, achats)
- Interface moderne et responsive

## Technologies utilisées
- Java 17+
- JavaFX
- MySQL
- Maven
- JasperReports pour les rapports
